# Generador de Búsquedas Aleatorias 🚀

Este es un script sencillo y eficaz diseñado para automatizar la exploración de contenidos en Bing. La herramienta genera frases de búsqueda variadas y las ejecuta de forma secuencial en una sola pestaña, optimizando el tiempo de navegación.

Este proyecto fue desarrollado íntegramente con la asistencia de Inteligencia Artificial.

## ✨ Funciones principales
- Búsqueda Continua: Realiza un ciclo de 30 búsquedas automáticas.
- Lógica Anti-Repetición: Genera combinaciones de frases aleatorias para que cada búsqueda sea única.
- Control Total: Incluye un botón para Iniciar y otro para Detener el proceso en cualquier momento.
- Contador Gigante: Un indicador visual claro para ver el progreso de la sesión.
- Intervalos Inteligentes: Pausas aleatorias de 10 a 16 segundos para simular un ritmo de navegación humano y fluido.

## 🛠️ Configuración rápida
Para que el programa funcione correctamente en una sola pestaña:
- Entra a Bing.com y abre el menú de configuración.
- En la sección Resultados, desactiva la opción: "Abrir vínculos de los resultados de búsqueda en una pestaña nueva".
- Guarda los cambios.

## 🚀 Instrucciones de uso en local
- Guarda el código en un archivo llamado generador.html.
- Abre ese archivo en tu navegador (Edge o Chrome).
- Haz clic en INICIAR.
- El programa abrirá una pestaña de Bing y actualizará el contador automáticamente hasta llegar a 30.
